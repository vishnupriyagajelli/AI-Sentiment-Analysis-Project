# AI Sentiment Analysis Project

This project classifies movie reviews as Positive or Negative using NLP and Machine Learning.

## Technologies Used
- Python
- NLTK
- Scikit-learn
- TF-IDF
- Logistic Regression

## Steps
- Data Cleaning & Preprocessing
- Text Vectorization using TF-IDF
- Model Training
- Model Evaluation
- Sentiment Prediction

## Result
Achieved ~90% accuracy in sentiment classification.
